# SentinelOps

**Système de veille stratégique et de réponse aux crises**

SentinelOps est une plateforme de cartographie collaborative permettant aux communautés affectées par des crises de signaler les dégâts en temps réel à l'aide de photos géolocalisées.

## Fonctionnalités

- 🗺️ Cartographie tactique en temps réel
- 📸 Transmission d'intelligence terrain
- 📡 Mode hors ligne avec synchronisation automatique
- 🌍 Support 6 langues ONU (🇫🇷 🇬🇧 🇪🇸 🇷🇺 🇨🇳 🇸🇦)
- 📊 Centre d'opérations unifié
- 🤖 Intelligence artificielle (AST - Analyse Stratégique Tactique)

## Stack technique

- Frontend: React + Vite + Tailwind + Leaflet
- Backend: FastAPI + PostgreSQL/PostGIS
- Conteneurisation: Docker
